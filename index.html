基于ubuntu server搭建一套家庭文件下载访问服务器

做这一套东西其实并非一时兴起，算的是早有预谋罢之前就尝试利用香橙派zero，来搭建个简单的网盘，当时部署了一个samba和qb前者利用smb连接我的电脑和home server，qb则作为bt下载来离线一些动画与资源，当时的这套东西并未做外网访问的设置 仅仅是在家里的内网运行了一段时间，后来发现稳定性不行就让他停了，让一个派带着2.5寸机械硬盘跑还是太难为他了，平均半天掉一次盘我也很怕硬盘gg，所以之后我也是拔掉硬盘用frp作为一个小型下bt离线器开始工作，直到这次我收到了一台朋友送的软路由。。。

配置如下

  • i5-2537m Q1’11的产品 大致时间是2010罢 基准频率1.4Ghz 最大睿频2.3Ghz 32nm TDP只有17w 可以说很省电了
  • 8g笔记本内存 没看牌子
  • 32G msata 32G sata 前者被我当作系统盘来用了 后者拆下来换成了我的1t的2.5寸机械盘

  1. 高自定义

》》》》》快进到系统安装完毕

第一步 部署samba

首先肯定是安装

apt-get install samba

一般情况下samba的配置文件会放在/etc/samba/下 那么肯定是备份一份conf

mv smb.conf smb.conf.bak

让后新建个空白的conf配置文件

nano smb.conf

先保存 推出 新建，或者基于你希望的用户加道用户组

smbpasswd -a root

之后 重新配置conf文件

配置

作用

browseable

valid users

允许访问的用户和组

完成之后 用

sudo systemctl restart nmbd.service

摘记 拾

1月 31 2022

除了除夕夜这天晚上有点要炸死年兽的气势，其他时候还真的是，没有一点年味。。

现在是九点三十 烟花停了大概半小时了

许久没写了，从学校到放假 到现在有很多事情 但真正能想到的已经没有几分了，更何况能写出来，

就这么过去了，这是我最大的感触，上学放学放假一气呵成 无论愿意与否 流畅至极，学校发生了许多事情，从学生会到个人再到学校，比赛，领导视察，还有些跟表演似的公开课，我真的不知道这些有何意义，当然这个学期下来我也发现我是真的不会带团队，诚然 老师说的是没有错的，任何大事不可能由一个人完成，我也承认这是我性格的问题 但是要怎么做在我这依旧是个谜，谁都没能给我答案也包括我

今年在学校不普通的收获之一或许便是拿到了学校部分计算机网络的权限吧，所有的希沃平板，几台电子班牌，一台服务器，其中大部分用ms17010便可以很轻松的拿到权限，当然除了那台录播室的服务器其他机器都开着远程端口且都没密码。。而且老师都不会查，拿来做个网络代理免费上网还是很好的

关于虚无主义，我知道那是魔鬼说的话，但是我真的不知道从何辩驳，我没办法像向浮士德一样说出“这是我生命中最崇高的瞬际” 反而是越来越无法否认魔鬼所言，我知道，我当然知道虚无主义是一件很可悲的想法，但是我该这么办呢，我当然知道要去防抗，我要去拒绝，我要让自己成为意义，但最后这意义似乎依旧在虚无中，我所能观察到最好的对抗他的方法，或者说对抗他的人便是那些为了生存疲于奔命的人，生存所带来的压力与痛苦远超于精神之上，甚至我有时候在想，我们的所思所想到底有何意义，相较于他们的痛苦我们到底算什么，我们到底在痛苦个什么鸡巴，或许真的像我妈所说不要去看那么多事情，没有意义，你解决不了， 对啊 解决不了的啊 振臂高呼吧 让呼声盖过饥肠辘辘的声音吧

方向 不会放弃这些思考，尽管可能让人发笑，尽管他的意义似乎我自己都开始怀疑，尽管有时候我甚至觉得他是顶级的网络过家家，

我想起了那张图 一个人与书 三个环境 鲜花 战火 朝阳 随着那人脚下的书越来越高他都看见了 ，令我不解的是在战火底下的朝阳有何意义
